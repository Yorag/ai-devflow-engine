# A4.9a ContextEnvelope Builder Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` as the default execution skill for this plan. Use `superpowers:executing-plans` only as a fallback when the environment cannot safely dispatch bounded subagents. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement `ContextSourceResolver` and `ContextEnvelopeBuilder` so AL05 can resolve stage-scoped context sources, construct canonical `ContextEnvelope` / `ContextManifest` objects, render final stage-call messages through the existing `PromptRenderer`, and persist `context_manifest` into `StageArtifact.process` by reusing the existing `ArtifactStore.append_process_record()` entrypoint only.

**Architecture:** Add a pure context-resolution module under `backend/app/context/source_resolver.py` and a bounded builder module under `backend/app/context/builder.py`. The resolver consumes already-frozen run snapshots, existing runtime records, stable references, and approved solution artifacts to create sectioned `ContextBlock` values. The builder composes those blocks with `PromptRenderer`, `ToolRegistry`, `ContextManifest.from_envelope()`, and the existing `ArtifactStore.append_process_record()` path. It does not add new shared `ArtifactStore` helpers, does not query logs as business truth, and does not widen into A4.9b compression or A4.9d runtime-loop behavior.

**Tech Stack:** Python 3.11+, Pydantic v2, SQLAlchemy runtime models as read inputs only, repository-local `uv`, existing `PromptRenderer`, existing `ArtifactStore`, and pytest via `uv run pytest`.

    def resolve_context_references(

        *,
        session_id: str,
        run_id: str,
        stage_run_id: str,
        stage_type: StageType,

        context_references: Sequence[ContextReference],

        built_at: datetime,
    ) -> ResolvedContextSources:

```

Resolver rules:

- Only consume explicit inputs passed to the resolver. Do not load more records from the database.
- Only include `StageArtifactModel`, `ClarificationRecordModel`, and `ApprovalDecisionModel` values whose `run_id` is in `allowed_context_run_ids`.
- For `code_generation`, `test_generation_execution`, and `code_review`, parse `process["solution_design_artifact"]` with `SolutionDesignArtifactRead.model_validate(...)` when present and emit an `input_artifact_refs` block summary that includes:
  - `implementation_plan.plan_id`
  - task ids
  - task order
  - dependency ids
- Ignore malformed artifact payloads instead of throwing resolver-wide exceptions.
- Do not read runtime logs or `log.db`.
- Represent user / human / tool / test / delivery observations as `ContextTrustLevel.UNTRUSTED_OBSERVATION`.
- Represent approved structured artifact refs and explicit `ContextReference` / `ChangeSet` refs as `ContextTrustLevel.TRUSTED_REFERENCE`.
- Keep `boundary_action` conservative:
  - `ALLOW` for approved structured artifact refs
  - `REFERENCE_ONLY` for `ChangeSet`, tool trace, command trace, and model/process refs
  - `QUARANTINE` for clarification answers, approval reasons, and free-text observations

class ContextEnvelopeBuilder:
    def build_for_stage_call(self, request: ContextBuildRequest) -> ContextBuildResult:

    def render_messages(
        self,
        *,
        envelope: ContextEnvelope,
        prompt_render_result: PromptRenderResult,

    def append_manifest_record(
        self,
        *,
        artifact_id: str,
        manifest: ContextManifest,
        trace_context: TraceContext,
    ) -> None:
        ...
```

Builder rules:

- Validate identity consistency before building:
  - `request.trace_context.session_id == request.session_id`
  - `request.trace_context.run_id == request.run_id`
  - `request.trace_context.stage_run_id == request.stage_run_id`
  - `request.template_snapshot.run_id == request.run_id`
  - `request.graph_definition.run_id == request.run_id`
  - `request.provider_snapshot.run_id == request.run_id`
  - `request.model_binding_snapshot.run_id == request.run_id`
  - the built `ContextEnvelope.model_binding_snapshot_ref` must equal `request.model_binding_snapshot.snapshot_id`
- Treat `request.template_version` as an already-frozen caller input. Do not derive it by reading the latest template or mutating `TemplateSnapshot`.
- If `request.model_call_type is ModelCallType.STRUCTURED_OUTPUT_REPAIR`, require `request.parse_error` and pass it into `PromptRenderRequest(parse_error=...)`.
- Use `request.allowed_context_run_ids` when provided; otherwise default it to `(request.run_id,)`.
- Read the current stage contract only from `request.graph_definition.stage_contracts[request.stage_type.value]`.
- Resolve the current stage role prompt only from `request.template_snapshot.stage_role_bindings`.
- Build `available_tools` only from `tool_registry.list_bindable_tools()` filtered by the current `allowed_tools` list.
- Use `PromptRenderer.render_messages(...)` for:
  - `runtime_instructions`
  - `stage_contract`
  - `agent_role_prompt`
  - `task_objective`
  - `specified_action`
  - `available_tools`
  - `response_schema`
- Do not change `PromptRenderer`. Instead, append the non-prompt sections to the final user-visible message in canonical section order:
  - `input_artifact_refs`
  - `context_references`
  - `working_observations`
  - `reasoning_trace`
  - `recent_observations`
- Final rendered output ref must describe the full context-envelope message sequence rather than the builtin-prompt-only render, for example:

```python
f"artifact://context-envelopes/{request.run_id}/{request.stage_run_id}/{request.model_call_type.value}"
```

- Final `render_hash` must be recomputed from the full rendered message sequence after those non-prompt sections are appended.
- `compression_trigger_token_threshold` must be computed only from the current frozen snapshots:

```python
floor(
    request.provider_snapshot.capabilities.context_window_tokens
    * request.runtime_limit_snapshot.context_limits.compression_threshold_ratio
)
```

- Persist the manifest only through the existing shared entry:

```python
self._artifact_store.append_process_record(
    artifact_id=artifact_id,
    process_key="context_manifest",
    process_value=manifest.model_dump(mode="json"),
    trace_context=trace_context,
)
```

- Do not add any new process helper, helper method on `ArtifactStore`, or `backend/app/services/artifacts.py` edit.

@dataclass(frozen=True, slots=True)
class ResolvedContextSources:
    context_references: tuple[ContextBlock, ...] = ()
    working_observations: tuple[ContextBlock, ...] = ()
    reasoning_trace: tuple[ContextBlock, ...] = ()
    recent_observations: tuple[ContextBlock, ...] = ()


class ContextSourceResolver:
    def resolve_stage_inputs(...):
        allowed = set(allowed_context_run_ids) or {run_id}
        blocks: list[ContextBlock] = []
        for artifact in stage_artifacts:
            if artifact.run_id not in allowed:
                continue
            solution = self._solution_design_artifact(artifact)
            if solution is None:
                continue
            if stage_type not in {
                StageType.CODE_GENERATION,
                StageType.TEST_GENERATION_EXECUTION,
                StageType.CODE_REVIEW,
            }:
                continue
            blocks.append(
                self._block(
                    block_id=f"implementation-plan:{solution.implementation_plan.plan_id}",
                    section=ContextEnvelopeSection.INPUT_ARTIFACT_REFS,
                    trust_level=ContextTrustLevel.TRUSTED_REFERENCE,
                    boundary_action=ContextBoundaryAction.ALLOW,
                    summary=self._implementation_plan_summary(solution),
                    content_ref=artifact.payload_ref,
                    sources=(
                        ContextSourceRef(
                            source_kind="solution_design_artifact",
                            source_ref=f"stage-artifact://{artifact.artifact_id}",
                            source_label=solution.artifact_id,
                            version_ref=solution.implementation_plan.plan_id,
                        ),
                    ),
                )
            )
        return tuple(blocks)

    def resolve_context_references(...):
        allowed = set(allowed_context_run_ids) or {run_id}
        return ResolvedContextSources(
            context_references=self._context_reference_blocks(context_references, built_at),
            working_observations=self._working_observation_blocks(stage_artifacts, change_sets, allowed),
            reasoning_trace=self._reasoning_trace_blocks(stage_artifacts, allowed),
            recent_observations=self._recent_observation_blocks(
                clarifications,
                approval_decisions,
                allowed,
            ),
        )
