# A4.8d PromptRenderer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` as the default execution skill for this plan. Use `superpowers:executing-plans` only as a fallback when the environment cannot safely dispatch bounded subagents. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement `PromptRenderer` so stage-execution and structured-output-repair calls render versioned builtin prompt assets, stage-contract truth, agent-role prompt text, tool descriptions, task intent, and response-schema metadata into traceable prompt sections plus a deterministic final message sequence.

**Architecture:** Add a pure rendering module at `backend/app/prompts/renderer.py` that consumes existing `PromptRegistry`, `TemplateSnapshot`, `GraphDefinition.stage_contracts`, `ToolBindableDescription`, and `PromptRenderMetadata` contracts without introducing a second source of truth for stage rules or tool permissions. Keep `ContextEnvelope` construction out of scope for this slice; instead, the renderer returns prompt-section metadata, message-sequence content, and stable hashes that A4.9a can later embed into `ContextBlock` / `ContextManifest`.

**Tech Stack:** Python 3.11+, Pydantic v2, repository-local `uv`, existing prompt/context schemas, and pytest via `uv run pytest`.

## Log And Audit Integration

This slice does not add an API route, runtime node, tool call, provider call, workspace mutation, configuration mutation, or command surface, so it does not need a new `Log & Audit Integration` implementation path.

Required guardrails for this slice:

- Rendering failures must surface as structured Python errors and must not fall back to inline hardcoded prompt text.
- Prompt asset metadata (`prompt_id`, `prompt_version`, `source_ref`, `cache_scope`, `content_hash`) must stay in renderer/context metadata and must not be injected into model-visible message content.
- The renderer must not treat the user-editable `agent_role_prompt` as a system-owned prompt asset and must not manufacture fake `PromptVersionRef` metadata for it.

## Design Decisions

- `PromptRenderer` will be a pure rendering boundary and will not instantiate `ContextBlock` or `ContextEnvelope`.
- `PromptRenderResult` will wrap `PromptRenderMetadata` so A4.9a can later reuse the same metadata object when constructing `ContextManifest` records.
- `stage_contract` will render from the current contract dict in a stable field order:
  - `stage_responsibility`
  - `input_contract`
  - `output_contract`
  - `structured_artifact_required`
  - `allowed_tools`
  - `runtime_limits`
  - optional `validation_pass`
- `tool_usage_template` remains the only builtin asset that shapes tool-usage phrasing; actual tool availability still comes from the request’s bindable tool list and must be validated against the stage contract’s `allowed_tools`.
- `PromptSectionRef.rendered_content_hash` remains the per-section render hash field. This slice will not rename it. `render_hash` in acceptance language maps to that existing field for section-level metadata and to `PromptRenderMetadata.rendered_prompt_hash` for the final message sequence.
- `ContextManifest.from_envelope()` will gain an explicit `system_prompt_ref` override so later A4.9a can store the real template-snapshot prompt source instead of incorrectly defaulting to the first builtin prompt asset.
- Final rendered refs use deterministic artifact-style strings:
  - section refs: `artifact://prompt-sections/<render-id>/<section-id>`
  - message-sequence ref: `artifact://rendered-messages/<render-id>`

```python
from __future__ import annotations

import json
from hashlib import sha256
from typing import Any

from pydantic import BaseModel, ConfigDict, Field, ValidationInfo, field_validator, model_validator

from backend.app.context.schemas import ContextEnvelopeSection, PromptSectionRef
from backend.app.domain.enums import StageType
from backend.app.prompts.definitions import (
    RUNTIME_INSTRUCTIONS_PROMPT_ID,
    STRUCTURED_OUTPUT_REPAIR_PROMPT_ID,
    TOOL_USAGE_TEMPLATE_PROMPT_ID,
)
from backend.app.prompts.registry import PromptAssetNotFoundError, PromptRegistry
from backend.app.schemas.prompts import (
    ModelCallType,
    PromptCacheScope,
    PromptRenderMetadata,
    PromptVersionRef,
)
from backend.app.tools.protocol import ToolBindableDescription


JsonObject = dict[str, Any]


class _StrictFrozenModel(BaseModel):
    model_config = ConfigDict(extra="forbid", frozen=True)


class PromptRendererError(RuntimeError):
    def __init__(
        self,
        error_code: str,
        message: str,
        *,
        prompt_id: str | None = None,
        stage_type: StageType | None = None,
    ) -> None:
        self.error_code = error_code
        self.prompt_id = prompt_id
        self.stage_type = stage_type
        self.message = message
        super().__init__(message)


class PromptRenderedMessage(_StrictFrozenModel):
    role: str = Field(pattern=r"^(system|user)$")
    content: str = Field(min_length=1)


class PromptRenderedSection(_StrictFrozenModel):
    section: ContextEnvelopeSection
    section_id: str = Field(min_length=1)
    title: str = Field(min_length=1)
    rendered_text: str = Field(min_length=1)
    rendered_content_ref: str = Field(min_length=1)
    rendered_content_hash: str = Field(pattern=r"^[0-9a-f]{64}$")
    prompt_ref: PromptVersionRef | None = None
    cache_scope: PromptCacheScope | None = None

    def as_prompt_section_ref(self) -> PromptSectionRef:
        if self.prompt_ref is None or self.cache_scope is None:
            raise ValueError("Only prompt-backed sections can be converted to PromptSectionRef")
        return PromptSectionRef(
            section_id=self.section_id,
            title=self.title,
            prompt_ref=self.prompt_ref,
            rendered_content_ref=self.rendered_content_ref,
            rendered_content_hash=self.rendered_content_hash,
            cache_scope=self.cache_scope,
        )


class PromptRenderRequest(_StrictFrozenModel):
    render_id: str = Field(min_length=1)
    created_at: datetime
    model_call_type: ModelCallType
    stage_type: StageType
    template_snapshot_ref: str = Field(min_length=1)
    template_version: str = Field(min_length=1)
    system_prompt_ref: str | None = Field(default=None, min_length=1)
    stage_contract_ref: str = Field(min_length=1)
    stage_contract: JsonObject
    agent_role_prompt: str | None = Field(default=None, min_length=1)
    task_objective: str = Field(min_length=1)
    specified_action: str | None = Field(default=None, min_length=1)
    response_schema: JsonObject
    tool_descriptions: tuple[ToolBindableDescription, ...] = Field(default_factory=tuple)
    repair_error: str | None = Field(default=None, min_length=1)
    repair_candidate_output: str | None = Field(default=None, min_length=1)
    context_manifest_ref: str | None = Field(default=None, min_length=1)
    compressed_context_block_ref: str | None = Field(default=None, min_length=1)

    @model_validator(mode="after")
    def validate_repair_fields(self) -> "PromptRenderRequest":
        if self.model_call_type is ModelCallType.STRUCTURED_OUTPUT_REPAIR:
            if self.repair_error is None or self.repair_candidate_output is None:
                raise ValueError(
                    "structured_output_repair requests require repair_error and repair_candidate_output"
                )
        return self


class PromptRenderResult(_StrictFrozenModel):
    rendered_output_ref: str = Field(min_length=1)
    sections: tuple[PromptRenderedSection, ...]
    messages: tuple[PromptRenderedMessage, ...]
    metadata: PromptRenderMetadata


class PromptRenderer:
    def __init__(self, registry: PromptRegistry) -> None:
        self._registry = registry

    def render_runtime_instructions(
        self,
        request: PromptRenderRequest,
    ) -> PromptRenderedSection:
        asset = self._get_asset(RUNTIME_INSTRUCTIONS_PROMPT_ID)
        return self._prompt_backed_section(
            request=request,
            section=ContextEnvelopeSection.RUNTIME_INSTRUCTIONS,
            section_id="runtime_instructions",
            title="Runtime Instructions",
            body=asset.sections[0].body,
            prompt_ref=self._prompt_ref(asset),
        )

    def render_stage_contract(
        self,
        request: PromptRenderRequest,
    ) -> PromptRenderedSection:
        contract = request.stage_contract
        required = (
            "stage_responsibility",
            "input_contract",
            "output_contract",
            "structured_artifact_required",
            "allowed_tools",
            "runtime_limits",
        )
        missing = [field for field in required if field not in contract]
        if missing:
            raise PromptRendererError(
                "stage_contract_invalid",
                f"stage contract is missing required fields: {missing}",
                stage_type=request.stage_type,
            )
        allowed_tools = contract["allowed_tools"]
        if not isinstance(allowed_tools, list) or not all(isinstance(item, str) for item in allowed_tools):
            raise PromptRendererError(
                "stage_contract_invalid",
                "stage contract allowed_tools must be a string list",
                stage_type=request.stage_type,
            )
        requested_tools = {tool.name for tool in request.tool_descriptions}
        if not requested_tools.issubset(set(allowed_tools)):
            extra = sorted(requested_tools - set(allowed_tools))
            raise PromptRendererError(
                "stage_contract_conflict",
                f"tool descriptions are not allowed by the stage contract: {extra}",
                stage_type=request.stage_type,
            )
        rendered = (
            f"Stage Responsibility: {contract['stage_responsibility']}\n"
            f"Input Contract: {contract['input_contract']}\n"
            f"Output Contract: {contract['output_contract']}\n"
            f"Structured Artifact Required: {contract['structured_artifact_required']}\n"
            f"Allowed Tools: {', '.join(allowed_tools) if allowed_tools else '(none)'}\n"
            f"Runtime Limits: {json.dumps(contract['runtime_limits'], ensure_ascii=True, sort_keys=True)}"
        )
        if "validation_pass" in contract:
            rendered += (
                "\nValidation Pass: "
                + json.dumps(contract["validation_pass"], ensure_ascii=True, sort_keys=True)
            )
        return self._dynamic_section(
            request=request,
            section=ContextEnvelopeSection.STAGE_CONTRACT,
            section_id="stage_contract",
            title="Stage Contract",
            body=rendered,
        )

    def render_tool_usage(
        self,
        request: PromptRenderRequest,
    ) -> PromptRenderedSection | None:
        if not request.tool_descriptions:
            return None
        asset = self._get_asset(TOOL_USAGE_TEMPLATE_PROMPT_ID)
        tool_lines = []
        for tool in sorted(request.tool_descriptions, key=lambda candidate: candidate.name):
            tool_lines.append(
                f"- {tool.name}: {tool.description} (schema_version={tool.schema_version})"
            )
        body = asset.sections[0].body + "\n\nAvailable Tools:\n" + "\n".join(tool_lines)
        return self._prompt_backed_section(
            request=request,
            section=ContextEnvelopeSection.AVAILABLE_TOOLS,
            section_id="available_tools",
            title="Available Tools",
            body=body,
            prompt_ref=self._prompt_ref(asset),
        )

    def render_structured_output_repair(
        self,
        request: PromptRenderRequest,
    ) -> PromptRenderedSection:
        asset = self._get_asset(STRUCTURED_OUTPUT_REPAIR_PROMPT_ID)
        body = (
            asset.sections[0].body
            + "\n\nSchema:\n"
            + json.dumps(request.response_schema, ensure_ascii=True, sort_keys=True, indent=2)
            + "\n\nValidation Error:\n"
            + request.repair_error
        )
        return self._prompt_backed_section(
            request=request,
            section=ContextEnvelopeSection.RUNTIME_INSTRUCTIONS,
            section_id="structured_output_repair",
            title="Structured Output Repair",
            body=body,
            prompt_ref=self._prompt_ref(asset),
        )

    def render_messages(
        self,
        request: PromptRenderRequest,
    ) -> PromptRenderResult:
        sections: list[PromptRenderedSection] = []
        prompt_refs: list[PromptVersionRef] = []
        system_chunks: list[str] = []
        user_chunks: list[str] = []

        if request.model_call_type is ModelCallType.STRUCTURED_OUTPUT_REPAIR:
            repair = self.render_structured_output_repair(request)
            schema_section = self._dynamic_section(
                request=request,
                section=ContextEnvelopeSection.RESPONSE_SCHEMA,
                section_id="response_schema",
                title="Response Schema",
                body=json.dumps(request.response_schema, ensure_ascii=True, sort_keys=True, indent=2),
            )
            sections.extend([repair, schema_section])
            prompt_refs.append(repair.prompt_ref)
            system_chunks.extend([repair.rendered_text, schema_section.rendered_text])
            user_chunks.append(request.repair_candidate_output)
        else:
            runtime = self.render_runtime_instructions(request)
            stage_contract = self.render_stage_contract(request)
            sections.extend([runtime, stage_contract])
            prompt_refs.append(runtime.prompt_ref)
            if request.agent_role_prompt is not None:
                sections.append(
                    self._dynamic_section(
                        request=request,
                        section=ContextEnvelopeSection.AGENT_ROLE_PROMPT,
                        section_id="agent_role_prompt",
                        title="Agent Role Prompt",
                        body=request.agent_role_prompt,
                    )
                )
            tool_usage = self.render_tool_usage(request)
            if tool_usage is not None:
                sections.append(tool_usage)
                prompt_refs.append(tool_usage.prompt_ref)
            sections.append(
                self._dynamic_section(
                    request=request,
                    section=ContextEnvelopeSection.TASK_OBJECTIVE,
                    section_id="task_objective",
                    title="Task Objective",
                    body=request.task_objective,
                )
            )
            if request.specified_action is not None:
                sections.append(
                    self._dynamic_section(
                        request=request,
                        section=ContextEnvelopeSection.SPECIFIED_ACTION,
                        section_id="specified_action",
                        title="Specified Action",
                        body=request.specified_action,
                    )
                )
            sections.append(
                self._dynamic_section(
                    request=request,
                    section=ContextEnvelopeSection.RESPONSE_SCHEMA,
                    section_id="response_schema",
                    title="Response Schema",
                    body=json.dumps(request.response_schema, ensure_ascii=True, sort_keys=True, indent=2),
                )
            )
            for section in sections:
                if section.section in {
                    ContextEnvelopeSection.RUNTIME_INSTRUCTIONS,
                    ContextEnvelopeSection.STAGE_CONTRACT,
                    ContextEnvelopeSection.AGENT_ROLE_PROMPT,
                    ContextEnvelopeSection.AVAILABLE_TOOLS,
                }:
                    system_chunks.append(section.rendered_text)
                else:
                    user_chunks.append(section.rendered_text)

        messages = (
            PromptRenderedMessage(role="system", content="\n\n".join(system_chunks)),
            PromptRenderedMessage(role="user", content="\n\n".join(user_chunks)),
        )
        render_hash = self.compute_render_hash(messages)
        metadata = PromptRenderMetadata(
            render_id=request.render_id,
            model_call_type=request.model_call_type,
            prompt_refs=[prompt_ref for prompt_ref in prompt_refs if prompt_ref is not None],
            rendered_prompt_hash=render_hash,
            section_order=[section.section_id for section in sections],
            template_snapshot_ref=request.template_snapshot_ref,
            stage_contract_ref=request.stage_contract_ref,
            tool_schema_version=(
                max((tool.schema_version for tool in request.tool_descriptions), default="tool-schema-v1")
            ),
            context_manifest_ref=request.context_manifest_ref,
            compressed_context_block_ref=request.compressed_context_block_ref,
            created_at=request.created_at,
        )
        return PromptRenderResult(
            rendered_output_ref=f"artifact://rendered-messages/{request.render_id}",
            sections=tuple(sections),
            messages=messages,
            metadata=metadata,
        )

    def compute_render_hash(
        self,
        messages: tuple[PromptRenderedMessage, ...],
    ) -> str:
        serialized = "\n".join(
            f"{message.role}:{message.content}" for message in messages
        )
        return sha256(serialized.encode("utf-8")).hexdigest()

    def _get_asset(self, prompt_id: str):
        try:
            return self._registry.get(prompt_id)
        except PromptAssetNotFoundError as exc:
            raise PromptRendererError(
                "prompt_asset_missing",
                f"required prompt asset is unavailable: {prompt_id}",
                prompt_id=prompt_id,
            ) from exc

    def _prompt_ref(self, asset) -> PromptVersionRef:
        return PromptVersionRef(
            prompt_id=asset.prompt_id,
            prompt_version=asset.prompt_version,
            prompt_type=asset.prompt_type,
            authority_level=asset.authority_level,
            cache_scope=asset.cache_scope,
            source_ref=asset.source_ref,
            content_hash=asset.content_hash,
        )

    def _prompt_backed_section(
        self,
        *,
        request: PromptRenderRequest,
        section: ContextEnvelopeSection,
        section_id: str,
        title: str,
        body: str,
        prompt_ref: PromptVersionRef,
    ) -> PromptRenderedSection:
        rendered_content_hash = sha256(body.encode("utf-8")).hexdigest()
        return PromptRenderedSection(
            section=section,
            section_id=section_id,
            title=title,
            rendered_text=body,
            rendered_content_ref=f"artifact://prompt-sections/{request.render_id}/{section_id}",
            rendered_content_hash=rendered_content_hash,
            prompt_ref=prompt_ref,
            cache_scope=prompt_ref.cache_scope,
        )

    def _dynamic_section(
        self,
        *,
        request: PromptRenderRequest,
        section: ContextEnvelopeSection,
        section_id: str,
        title: str,
        body: str,
    ) -> PromptRenderedSection:
        rendered_content_hash = sha256(body.encode("utf-8")).hexdigest()
        return PromptRenderedSection(
            section=section,
            section_id=section_id,
            title=title,
            rendered_text=body,
            rendered_content_ref=f"artifact://prompt-sections/{request.render_id}/{section_id}",
            rendered_content_hash=rendered_content_hash,
        )
```
