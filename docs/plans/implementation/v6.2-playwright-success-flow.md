First browser-backed run failed because the route fixture matched `**/api/**`, which also intercepted Vite source modules such as `/src/api/hooks.ts`. The fixture returned a mocked API 404 for that module, React did not mount, and the first visible-region assertion failed. The route fixture was narrowed to requests whose pathname starts with `/api/`, preserving frontend module loading while still mocking backend API calls.
