- Do not create `log.db` SQLAlchemy models, migrations, repositories, JSONL writers, redaction services, audit services, runtime log emission, or log query endpoints.

- Do not define `GET /api/log-payloads/{payloadId}` or any full payload retrieval schema.
- Query schemas must be read-only filters and must reject mutation/replay/delete style extra fields.

## Log & Audit Integration

- Runtime log category, audit action, associated objects, and failure result are represented by schema fields only in this slice.
- `request_id`, `trace_id`, `correlation_id`, `span_id`, and `parent_span_id` are required or preserved by `TraceContext` and log projection schemas.
- Sensitive payload handling is represented by `payload_excerpt`, `payload_size_bytes`, and `redaction_status`; full raw payload fields are intentionally rejected.
- Log writer, audit writer, `log.db` index, and failure behavior are not implemented here; they remain owned by L1.2 and L2.3-L2.4/L4.1.
- Tests must prove log/audit projections do not replace `FeedEntry`, `StageInspectorProjection`, or `SessionEvent` product facts.

```python
from __future__ import annotations

from datetime import datetime
from typing import Any, Self

from pydantic import BaseModel, ConfigDict, Field


_INHERITED_TRACE_FIELDS = frozenset({"request_id", "trace_id", "correlation_id"})


class TraceContext(BaseModel):
    model_config = ConfigDict(extra="forbid")

    request_id: str = Field(min_length=1)
    trace_id: str = Field(min_length=1)
    correlation_id: str = Field(min_length=1)
    span_id: str = Field(min_length=1)
    parent_span_id: str | None = Field(default=None, min_length=1)
    session_id: str | None = Field(default=None, min_length=1)
    run_id: str | None = Field(default=None, min_length=1)
    stage_run_id: str | None = Field(default=None, min_length=1)
    approval_id: str | None = Field(default=None, min_length=1)
    tool_confirmation_id: str | None = Field(default=None, min_length=1)
    delivery_record_id: str | None = Field(default=None, min_length=1)
    graph_thread_id: str | None = Field(default=None, min_length=1)
    created_at: datetime

    def child_span(
        self,
        *,
        span_id: str,
        created_at: datetime,
        **object_updates: Any,
    ) -> Self:
        if inherited_updates := _INHERITED_TRACE_FIELDS.intersection(object_updates):
            names = ", ".join(sorted(inherited_updates))
            raise ValueError(
                f"child span cannot override inherited trace fields: {names}"
            )

        data = self.model_dump()
        data.update(object_updates)
        data["parent_span_id"] = self.span_id
        data["span_id"] = span_id
        data["created_at"] = created_at
        return self.__class__.model_validate(data)


__all__ = ["TraceContext"]
```
