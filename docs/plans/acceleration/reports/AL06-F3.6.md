## Review

Spec / plan compliance review result:

- Initial review requested changes.
- Important finding: Inspector state was not reset when the selected session or project changed, so the right rail could retain a stale target across workspace navigation.
- Fix result: `WorkspaceShell` now closes Inspector on explicit session changes and project changes, while leaving the same-session query refresh effect intact.
- Spec compliance re-review approved with no Critical, Important, or Minor findings.

Code quality / testing / regression review result:

- Initial review requested changes.
- Important finding: the tool-confirmation `Details` trigger sat too close in hierarchy and button weight to `Allow this execution` / `Deny this execution`.
- Minor finding: Escape close only worked while focus remained inside the Inspector panel.
- Fix result: the tool-confirmation details trigger now uses `inspector-trigger--quiet` with lower emphasis and separation from the primary actions, and `InspectorPanel` now closes from document-scope Escape while open.
- Code quality re-review approved. No Critical or Important findings remained.
